# React Fullstack

"React Fullstack" is a collection of libraries for creating fullstack applications based on React!

## Packages

- [Fullstack](packages/fullstack) — `npm i @react-fullstack/fullstack`
- [Render](packages/react-render-null) — `npm i @react-fullstack/render`

## Transport Plugins

- [Bun WebSocket Server](plugins/fullstack/bun-ws-server) — `npm i @react-fullstack/fullstack-bun-ws-server`
- [Bun WebSocket Client](plugins/fullstack/bun-ws-client) — `npm i @react-fullstack/fullstack-bun-ws-client`
- [Socket.io Server](plugins/fullstack/socket-server) — `npm i @react-fullstack/fullstack-socket-server`
- [Socket.io Client](plugins/fullstack/socket-client) — `npm i @react-fullstack/fullstack-socket-client`
